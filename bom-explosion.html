▶

item_bom

Parent-child BOM links
  • parent_item_guid Upper-level assembly GUID FK
  • bom_item_guid Lower-level component GUID FK
  • quantity Quantity per parent
  • uom Unit of measure
Traversal Rule
To find the next layer: take the current layer's bom_item_guid values and look them up in parent_item_guid to find their children. Repeat until no more children exist.

Edge Cases Handled

Circular References

      
Path tracking prevents infinite loops when BOM data contains cycles.

    

    

      
📏

      
Max Depth Guard

      
Hard limit of 50 levels stops runaway recursion on deep hierarchies.

    

    

      
🔗

      
Shared Components

      
Same child appearing under multiple parents is handled correctly.

    

    

      
🍃

      
Leaf Nodes

      
Components with no children naturally terminate the recursion.

    

    

      
🚫

      
Missing Records

      
LEFT JOIN ensures orphan GUIDs don't break the explosion query.
